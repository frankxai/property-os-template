# Success Criteria

The template is ready for a real owner when these gates pass.

## Install Gate

- `npm run validate` passes.
- Property JSON matches the schema.
- Knowledge records have approval status.
- Data boundaries are reviewed.
- Portal wiring notes are understood.

## Property Readiness Gate

- property name and public area are approved
- each unit has status and availability note
- amenities are verified
- house rules are reviewed
- photo rights are confirmed
- at least 8 FAQ answers are drafted
- urgent escalation path is documented privately

## Listing Gate

- own website draft exists
- Kleinanzeigen draft exists
- ImmoScout24 draft exists
- Immowelt draft exists
- missing facts are listed
- owner checklist is complete
- no listing is auto-published

## Renter Portal Gate

- property page has approved media or marked sample media
- inquiry form works
- renter portal has approved stay/move-in information
- support form works
- urgent path is clear
- no access secrets are committed

## Premium Gate

- first viewport shows the property, not generic SaaS
- copy is precise and calm
- no fake AI promises
- no text overlap on mobile
- owner approval states are visible
- sample data is clearly marked before production

## Commercial Gate

- a new owner can understand the template in under 20 minutes
- setup can be completed in under 60 minutes with sample data
- docs explain what is manual in v1
- paid install path is obvious
