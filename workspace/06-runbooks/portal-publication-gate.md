# Portal Publication Gate

Run this gate before showing a portal to renters or publishing a production Vercel deployment.

- `npm run validate` passes in the owner workspace
- portal content builds
- inquiry form tested
- support form tested
- mobile page inspected
- property image rights confirmed
- no sample-only copy remains in production
- no access secrets are exposed
- urgent escalation path is clear
- owner has approved renter-facing copy
