# Support

## Community Template

The free template includes docs, sample data, validation scripts, agent profiles, and MCP dry-run tooling. Community users can fork it, adapt it, and run the local checks.

## Partner Implementations

Partners can sell:

- owner onboarding and property intake
- premium portal setup
- private database and auth wiring
- GitHub, Vercel, Railway, email, and monitoring setup
- channel-specific listing workflows
- managed weekly review and support triage

## Boundaries

This repository does not provide legal, tax, tenant-screening, property-management, or emergency-response services. Production operators own local compliance, renter communication, data protection, and maintenance escalation.
