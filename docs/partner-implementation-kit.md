# Partner Implementation Kit

## Partner Promise

Help owners and agencies launch a cleaner, safer, more premium property operating system without replacing their entire property management stack.

## Deliverables

- owner intake
- property profile
- renter knowledge base
- portal wiring
- listing draft studio
- support and inquiry workflows
- visual QA
- owner training
- managed operations proposal

## Offer Ladder

| Offer | Best For | Revenue Model |
| --- | --- | --- |
| Starter Install | owner with one property | fixed setup |
| Premium Install | owner with media/copy needs | higher setup |
| Agency Workspace | boutique agency | setup plus license |
| Integration Sprint | operator with proven workflow | scoped project |
| Managed Ops | owner or agency | monthly retainer |

## Sales Script

Lead with:

- fewer repeated renter questions
- better property presentation
- owner-approved AI drafts
- cleaner listing workflow
- urgent support visibility

Do not promise:

- occupancy improvement
- legal compliance
- rent increase
- automatic publication
- tenant selection

## Handoff Proof

- validation output
- privacy scan output
- portal preview
- visual QA notes
- first listing drafts
- weekly review checklist
