# User Flows

## Owner

1. Fill property profile.
2. Review data boundaries.
3. Add approved facts and FAQs.
4. Generate listing drafts.
5. Review inquiry/support drafts.
6. Run weekly owner dashboard.

## Renter

1. Opens property page or renter portal.
2. Reads approved information.
3. Submits inquiry or support request if needed.
4. Receives no automated commitments in v1.

## Implementer

1. Forks template.
2. Runs validation.
3. Creates private owner workspace.
4. Wires portal template.
5. Runs release gates.
6. Offers monthly managed operations.

## Agent Swarm

1. Property Steward checks source facts.
2. Listing Ops drafts channel copy.
3. Inquiry Concierge drafts safe reply.
4. Maintenance Triage classifies support.
5. Privacy Compliance Reviewer blocks unsafe output.
6. Platform Architect handles hosted runtime.
7. Visual QA Agent checks portal readiness.
8. Implementer Growth Agent packages the offer.
