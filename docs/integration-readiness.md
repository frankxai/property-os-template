# Integration Readiness

## Rule

This template starts as manual-first. Integrations are added only after the owner workflow proves useful and the official access path is understood.

## Ready Now

- Own website publication from approved facts.
- Manual listing copy for Kleinanzeigen, ImmoScout24, and Immowelt.
- Renter portal knowledge from approved FAQ records.
- Inquiry and support intake through the portal template.
- Sanitized GitHub issue summaries.

## Dry-Run Next

- EstateSync payload mapping.
- ImmoScout24 required-field checklist.
- Resend notification templates.
- Cal.com viewing request flow.
- Postgres runtime storage.

## Later

- WhatsApp renter messaging.
- Stripe billing.
- E-signature.
- Accounting export.
- Vendor routing.

## Blocks

- No scraper-based posting.
- No automatic listing publication.
- No automatic tenant selection.
- No rent, deposit, or payment handling without legal/accounting review.
- No AI-generated legal advice.
