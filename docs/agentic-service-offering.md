# Agentic Service Offering

## Position

Property OS can become an agentic-as-a-service offering for owners, agencies, and implementers. The core commercial promise is not "AI replaces the landlord." The promise is a premium property operating layer that turns messy owner knowledge into a self-service renter experience, safer agent drafts, cleaner listings, and a repeatable install business.

## What Customers Buy

### Owners

- premium property website
- renter portal
- inquiry and support intake
- listing draft studio
- weekly owner review
- reduced repeated questions
- implementation readiness report before production handoff

### Agencies

- multi-owner operating pattern
- white-label setup
- team workflows
- support and vacancy visibility
- repeatable listing quality

### Implementers

- GitHub/Vercel/Railway template
- agent profiles and skills
- MCP architecture
- install runbooks
- proposal and retainer model
- portal and MCP readiness checks for every paid install

## Service Packages

| Package | Buyer | Deliverable | Revenue |
| --- | --- | --- | --- |
| Template Kit | implementer | repo template, docs, validation, sample portal | license or one-time purchase |
| Guided Owner Install | small owner | one property, renter portal, listing drafts, release gate | setup fee |
| Premium Property OS | owner/agency | custom portal, database, email, weekly review, visual QA | setup plus monthly retainer |
| Agency Partner Kit | agency | white-label install path, training, multi-owner docs | license plus per-install fee |
| Managed Ops | owner/agency | monthly FAQ, listing refresh, support triage, integration monitoring | recurring retainer |

## Why It Can Win

- Standard PMS tools are heavy and accounting-centered.
- Website builders lack property operations and approval gates.
- Chatbots are risky without approved knowledge and owner control.
- Notion/Softr templates do not provide code, portal, agent, MCP, and deployment substrate together.
- This template gives implementers a business, not just a project.

## Non-Negotiables

- Owner-approved decisions stay human.
- Integrations start manual or dry-run.
- Runtime data stays in secure storage.
- Public templates stay sample-safe.
- Every agent action is auditable.
