# Taste Standard

The template should feel premium because it is useful, precise, and calm.

## Owner Workspace

- short files
- clear headings
- status before prose
- missing facts visible
- approval gates explicit
- no decorative complexity

## Renter Experience

- real property media whenever possible
- no vague claims
- practical answers before brand language
- urgent path always findable
- mobile-first support flows
- approved facts only

## Listing Copy

Good listing copy is specific:

- what the property is
- who it fits
- what is included
- what still requires confirmation
- where the renter should inquire

Avoid:

- inflated luxury language without proof
- invented proximity claims
- hidden caveats
- pricing ambiguity once public
- terms that conflict with channel rules

## Visual Direction

Use a quiet, high-trust palette with enough contrast and warmth. The design should feel more like a boutique property operator than a SaaS landing page.

Cards are for repeated items, tickets, and tools. Do not put the whole page inside a card. Do not use generic abstract blobs or one-note purple gradients. Use property photos, plans, maps, or real operational status.
