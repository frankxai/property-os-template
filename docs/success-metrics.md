# Success Metrics

## Template Readiness

- validation passes
- privacy scan passes
- MCP smoke passes
- required agent profiles exist
- partner docs exist
- starter content is sample-safe

## Owner Success

- first property facts gathered in under 60 minutes
- top recurring questions drafted into FAQs
- listing drafts name missing facts
- weekly owner dashboard takes under 30 minutes

## Renter Success

- common answers are self-service
- support route is clear
- urgent path is visible
- no private access or payment data leaks

## Implementer Success

- install path is clear
- offer ladder is explainable
- excluded scope is explicit
- validation output can be shown as handoff proof
- monthly retainer tasks are defined
