# PRD Lite

## Product

Property OS Template is a public-safe operating workspace for owners, agencies, and implementers who want to launch an owner-approved property intelligence system.

## Outcomes

- Owner can add approved property facts.
- Renter can self-serve common information.
- Agent drafts stay owner-reviewed.
- Listing drafts are channel-ready but not auto-published.
- Implementer can fork, validate, install, and sell a managed service.

## Required Surfaces

- property profile
- approved knowledge base
- listing draft studio
- inquiry draft and approval
- maintenance triage
- vacancy pipeline
- weekly owner dashboard
- agent run audit
- MCP dry-run server
- Vercel portal wiring
- hosted runtime checklist
- partner implementation guide

## Release Criteria

- `npm run validate` passes
- privacy scan passes
- MCP smoke passes
- public examples are sample-safe
- owner approval rules are visible
- partner can explain excluded scope
