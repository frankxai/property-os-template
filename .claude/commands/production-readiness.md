# Production Readiness

Review whether a Property OS install is ready for paid usage or public launch.

Check:

- privacy scan and data boundaries
- schema validation
- visual QA on desktop and mobile
- build, typecheck, smoke, and deployment preview
- owner approval routes
- runtime database and retention
- email and notification safety
- integration dry-run evidence
- support and urgent escalation path
- implementer handoff docs

Return: ready, conditional, or blocked.
