# Agent Run Audit

Audit recent agent work before owner or client handoff.

1. Read `workspace/04-scorecards/weekly-owner-dashboard.md`.
2. Review agent outputs against `docs/agent-team.md` and `.claude/agents/`.
3. Confirm every consequential output has an owner action.
4. Check for private data, invented facts, pricing, availability, legal, publication, access, and urgent repair commitments.
5. Output findings as pass/block, required edits, and residual risk.
