# Install Client

Create a client install plan from this template.

1. Read `README.md`, `SUCCESS_CRITERIA.md`, `workspace/02-data-boundaries.md`, and `docs/integration-readiness.md`.
2. Choose owner, portfolio, agency, or technical partner mode.
3. List required property facts, media rights, renter FAQs, support rules, and listing channels.
4. Create a 14-day setup plan.
5. Identify required portal, database, email, storage, and deployment decisions.
6. End with owner approvals and blocked automations.
